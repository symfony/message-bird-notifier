MessageBird Notifier
====================

Provides [MessageBird](https://www.messagebird.com/) integration for Symfony Notifier.

DSN example
-----------

```
MESSAGEBIRD_DSN=messagebird://TOKEN@default?from=FROM
```

where:
- `TOKEN` is your MessageBird token
- `FROM` is your sender
